token_name

(PHP 4 >= 4.2.0, PHP 5)

token_name — Retourne le nom d'un élément de code source

Description

string token_name ( int $token )

token_name() retourne le nom symbolique correspondant à la valeur d'un token PHP.

Liste de paramètres

token

La valeur du token.

Valeurs de retour

Le nom symbolique du token donné.

Exemples

Exemple #1 Exemple avec token_name()

<?php
// 260 est la valeur token pour le token T_EVAL
echo token_name(260);        // -> "T_EVAL"

// une constante token renvoit vers son propre nom
echo token_name(T_FUNCTION); // -> "T_FUNCTION"
?>
