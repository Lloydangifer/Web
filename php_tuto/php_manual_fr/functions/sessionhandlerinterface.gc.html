SessionHandlerInterface::gc

(PHP 5 >= 5.4.0)

SessionHandlerInterface::gc — Nettoie les vieilles sessions

Description

abstract public bool SessionHandlerInterface::gc ( string $maxlifetime )

Nettoie les vieilles sessions expirées. Appelée par session_start(), en fonction de session.gc_divisor, session.gc_probability et session.gc_lifetime.

Liste de paramètres

maxlifetime

Les sessions non mises à jour durant les maxlifetime dernières secondes seront supprimées.

Valeurs de retour

La valeur retournée (habituellement TRUE en cas de succès, FALSE si une erreur survient). Notez que cette valeur est retournée en interne à PHP pour analyse.
