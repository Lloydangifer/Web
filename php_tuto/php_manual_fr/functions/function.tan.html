tan

(PHP 4, PHP 5)

tan — Tangente

Description

float tan ( float $arg )

Retourne la tangente de arg (arg en radians).

Liste de paramètres

arg

L'argument à traiter, en radians

Valeurs de retour

La tangente de arg

Exemples

Exemple #1 Exemple avec tan()

<?php

echo tan(M_PI_4); // 1

?>
