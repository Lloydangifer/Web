filter_id

(PHP 5 >= 5.2.0)

filter_id — Retourne l'identifiant d'un filtre nommé

Description

int filter_id ( string $filtername )

Liste de paramètres

filtername

Nom du filtre à récupérer.

Valeurs de retour

Identifiant du filtre en cas de succès, ou FALSE si le filtre n'existe pas.
