mysqli_fetch

(PHP 5 < 5.4.0)

mysqli_fetch — Alias de mysqli_stmt_fetch()

Description

mysqli_fetch() est un alias de la fonction mysqli_stmt_fetch().

Avertissement

Cette fonction a devenue OBSOLETE depuis PHP 5.3.0 et a été SUPPRIMEE depuis PHP 5.4.0.
