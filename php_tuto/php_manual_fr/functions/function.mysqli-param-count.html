mysqli_param_count

(PHP 5 < 5.4.0)

mysqli_param_count — Alias de mysqli_stmt_param_count()

Description

mysqli_param_count() est un alias de la fonction mysqli_stmt_param_count().

Avertissement

Cette fonction a devenue OBSOLETE depuis PHP 5.3.0 et a été SUPPRIMEE depuis PHP 5.4.0.
