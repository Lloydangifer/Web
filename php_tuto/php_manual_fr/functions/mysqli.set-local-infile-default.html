mysqli_set_local_infile_default

(PHP 5)

mysqli::set_local_infile_default -- mysqli_set_local_infile_default — Rétablit le gestionnaire par défaut pour la commande LOAD LOCAL INFILE

Description

void mysqli_set_local_infile_default ( mysqli $link )

Désactive un gestionnaire LOAD DATA INFILE LOCAL précédemment définit avec la fonction mysqli_set_local_infile_handler().

Liste de paramètres

link

Seulement en style procédural : Un identifiant de lien retourné par la fonction mysqli_connect() ou par la fonction mysqli_init()

Valeurs de retour

Aucune valeur n'est retournée.

Exemples

Voir l'exemple de la fonction mysqli_set_local_infile_handler()
