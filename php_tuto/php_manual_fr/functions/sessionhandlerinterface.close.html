SessionHandlerInterface::close

(PHP 5 >= 5.4.0)

SessionHandlerInterface::close — Ferme la session

Description

abstract public bool SessionHandlerInterface::close ( void )

Ferme la session courante. Cette fonction est appelée automatiquement lors de la fermeture de la session, ou explicitement via session_write_close().

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La valeur retournée (habituellement TRUE en cas de succès, FALSE si une erreur survient). Notez que cette valeur est retournée en interne à PHP pour analyse.
