atan

(PHP 4, PHP 5)

atan — Arc tangente

Description

float atan ( float $arg )

Retourne l'arc tangent de arg, en radians. atan() est la fonction inverse de tan(), ce qui signifie que a==tan(atan(a)) pour toute valeur qui soit dans l'intervalle de validité de atan().

Liste de paramètres

arg

L'argument à traiter

Valeurs de retour

L'arc tangent de arg, en radians.
