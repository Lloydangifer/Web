session_register_shutdown

(PHP >=5.4.0)

session_register_shutdown — Fonction de fermeture de session

Description

void session_register_shutdown ( void )

Enregistre la fonction session_write_close() comme fonction de fermeture de session.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Erreurs / Exceptions

Émet une alerte de type E_WARNING si l'enregistrement de la fonction de fermeture échoue.
