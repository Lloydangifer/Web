session_encode

(PHP 4, PHP 5)

session_encode — Encode les données de session

Description

string session_encode ( void )

session_encode() retourne une chaîne linéarisée contenant les variables de la session courante encodées stockées dans la variable superglobale $_SESSION.

Notez que la méthode de linéarisation n'est pas la même que celle utilisée par la fonction serialize(). Cette méthode de linéatisation est interne à PHP et peut être définie en utilisant l'option de configuration session.serialize_handler.

Valeurs de retour

Retourne le contenu encodé de la session courante.
