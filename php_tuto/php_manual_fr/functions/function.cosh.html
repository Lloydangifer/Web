cosh

(PHP 4 >= 4.1.0, PHP 5)

cosh — Cosinus hyperbolique

Description

float cosh ( float $arg )

Retourne le cosinus hyperbolique de arg, défini comme (exp(arg) + exp(-arg))/2.

Liste de paramètres

arg

L'argument à traiter

Valeurs de retour

Le cosinus hyperbolique de arg.
