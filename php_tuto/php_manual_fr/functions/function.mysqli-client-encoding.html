(PHP 5 < 5.4.0)

mysqli_client_encoding — Alias de mysqli_character_set_name()

Description

Cette fonction est un alias de la fonction mysqli_character_set_name().

Avertissement

Cette fonction a devenue OBSOLETE depuis PHP 5.3.0 et a été SUPPRIMEE depuis PHP 5.4.0.
