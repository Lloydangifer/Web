mysqli_stmt::next_result -- mysqli_stmt_next_result — Lit le résultat suivant depuis une requête multiple

Description

Style orienté objet (method):

public bool mysqli_stmt::next_result ( void )

Style procédural:

bool mysqli_stmt_next_result ( mysql_stmt $stmt )

Lit le résultat suivant depuis une requête multiple.

Liste de paramètres

stmt

Style procédural uniquement : Un identifiant de requête retourné par la fonction mysqli_stmt_init().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Erreurs / Exceptions

Lance une alerte de niveau E_STRICT s'il n'existe pas de jeu de résultat, et suggère d'utiliser la méthode mysqli_stmt::more_results() avant d'appeler mysqli_stmt::next_result() dans ce cas là.

Voir aussi

  • mysqli_stmt::more_results() - Vérifie s'il y a encore des résultats depuis une requête multiple
  • mysqli::multi_query() - Exécute une requête MySQL multiple
