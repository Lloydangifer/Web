sinh

(PHP 4 >= 4.1.0, PHP 5)

sinh — Sinus hyperbolique

Description

float sinh ( float $arg )

Retourne le sinus hyperbolique de arg, défini comme (exp(arg) - exp(-arg))/2.

Liste de paramètres

arg

L'argument à traiter

Valeurs de retour

Le sinus hyperbolique de arg.
