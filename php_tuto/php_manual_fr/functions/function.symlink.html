symlink

(PHP 4, PHP 5)

symlink — Crée un lien symbolique

Description

bool symlink ( string $target , string $link )

symlink() crée un lien symbolique pour l'objet target avec le nom de link.

Liste de paramètres

target

La cible du lien.

link

Le nom du lien.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Historique

Version Description
5.3.0 Cette fonction est maintenant disponible sous Windows (Vista, Server 2008 ou plus récent).

Exemples

Exemple #1 Création d'un lien symbolique

<?php
$target  = 'uploads.php';
$link = 'uploads';
symlink($target, $link);

echo  readlink($link);
?>

Notes

Note: Pour Windows uniquement : cette fonction n'est disponible que pour Windows Vista/Windows Server 2008 ou supérieur. Pour les versions antérieures, cette fonction ne supporte pas les liens symboliques.
