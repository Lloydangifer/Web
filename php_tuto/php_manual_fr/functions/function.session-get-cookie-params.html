session_get_cookie_params

(PHP 4, PHP 5)

session_get_cookie_params — Lit la configuration du cookie de session

Description

array session_get_cookie_params ( void )

Lit la configuration du cookie de session.

Valeurs de retour

Retourne un tableau, qui contient les éléments suivants :

  • "lifetime" : durée de vie du cookie.
  • "path" : le chemin où les informations sont stockées.
  • "domain" : le domaine du cookie.
  • "secure" : le cookie ne doit être envoyé que sur des connexions sécurisées.
  • "httponly" : le cookie ne sera accessible que via le protocole HTTP.

Historique

Version Description
5.2.0 L'entrée "httponly" a été ajouté dans le tableau retourné.
4.0.4 L'entrée "secure" a été ajouté dans le tableau retourné.
