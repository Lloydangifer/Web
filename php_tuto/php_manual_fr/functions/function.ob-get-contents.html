ob_get_contents

(PHP 4, PHP 5)

ob_get_contents — Retourne le contenu du tampon de sortie

Description

string ob_get_contents ( void )

Retourne le contenu du tampon de sortie sans l'effacer.

Valeurs de retour

Retourne le contenu du tampon de sortie sans l'effacer ou FALSE, si la tamporisation de sortie n'est pas activ√©e.

Exemples

Exemple #1 Exemple avec ob_get_contents()

<?php

ob_start ();

echo  "Bonjour ";

$out1 = ob_get_contents();

echo  "le monde !";

$out2 = ob_get_contents();

ob_end_clean();

var_dump($out1, $out2);
?>

L'exemple ci-dessus va afficher :

string(8) "Bonjour "
string(18) "Bonjour le monde !"
