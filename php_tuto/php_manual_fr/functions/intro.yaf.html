Introduction

L'extension Yet Another Framework(Yaf) est un framework PHP, utilisé pour le développement d'application web. Elle fournit une interface orientée objet à PHP pour le développement d'application web.

Yaf requière une version PHP supérieur à 5.2.1, et ne pourra pas fonctionner avec les versions antérieures.
