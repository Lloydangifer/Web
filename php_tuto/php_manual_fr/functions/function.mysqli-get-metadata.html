mysqli_get_metadata

(PHP 5 < 5.4.0)

mysqli_get_metadata — Alias de mysqli_stmt_result_metadata()

Description

mysqli_get_metadata() est un alias de la fonction mysqli_stmt_result_metadata().

Avertissement

Cette fonction a devenue OBSOLETE depuis PHP 5.3.0 et a été SUPPRIMEE depuis PHP 5.4.0.
