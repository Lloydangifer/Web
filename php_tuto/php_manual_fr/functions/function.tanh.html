tanh

(PHP 4 >= 4.1.0, PHP 5)

tanh — Tangente hyperbolique

Description

float tanh ( float $arg )

Retourne la tangente hyperbolique de arg, définie comme sinh(arg)/cosh(arg).

Liste de paramètres

arg

L'argument à traiter

Valeurs de retour

La tangente hyperbolique de arg.
