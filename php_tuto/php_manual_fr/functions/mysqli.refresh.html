mysqli::refresh -- mysqli_refresh — Rafraîchie

Description

Style orienté objet

public bool mysqli::refresh ( int $options )

Style procédural

int mysqli_refresh ( resource $link , int $options )

Rafraîchie les tables ou les caches, ou ré-initialise les informations du serveur de réplication.

Liste de paramètres

link

Seulement en style procédural : Un identifiant de lien retourné par la fonction mysqli_connect() ou par la fonction mysqli_init()

options

Les options de rafraîchissement, en utilisant les constantes MYSQLI_REFRESH_* telles que documentées dans la documentation sur les constantes MySQLi.

Voir aussi la documentation officielle sur » le rafraîchissement en MySQL.

Valeurs de retour

TRUE si le rafraîchissement s'est déroulé avec succès, FALSE sinon.
